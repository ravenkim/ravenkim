# 자기소개서

## 소개

AI 도구를 적극 활용해 생산성을 높이는 React 프론트엔드 개발자 김해준입니다.

React를 주력으로 3년 이상의 실무 경험을 쌓았습니다. AI 챗봇, LLM 시각화 플랫폼 등 AI 서비스의 프론트엔드를 처음부터 설계하고 구현한 경험이 있으며, Claude Code와 Cursor 같은 AI 코딩 도구를 실무에 적극 도입해 팀 전체의 개발 속도를 높여왔습니다. 단순히 기능을 구현하는 데 그치지 않고, 구조적이고 확장 가능한 코드 아키텍처와 빠른 개발 사이클을 동시에 추구합니다.

---

## AI 개발 방식

AI를 단순한 코드 생성기가 아닌 구조적 실행 파트너로 활용합니다. 탐색 → 계획 → 실행 → 검증의 4단계 사이클을 AI와의 협업에도 동일하게 적용하고, 큰 작업은 Phase 단위로 분할해 컨텍스트 오염 없이 단계별로 진행합니다. 팀 규칙과 디자인 시스템은 `.cursor/rules` 파일로 관리해 AI가 새 코드를 생성할 때도 일관된 기준을 따르도록 구성합니다. 무엇을 만들지보다 언제 완성됐는지를 먼저 정의하는 방식으로 협업의 완성도를 높입니다.

## 핵심 강점

### AI 서비스 프론트엔드 구현

LLM·AI 기반 서비스의 프론트엔드를 처음부터 설계하고 구현한 경험이 있습니다. LimeLab LLM 시각화 공통 템플릿에서는 스트리밍 응답 처리와 디자인 시스템을 도입했고, Bllossom 모델을 연동한 LLM VOC 플랫폼과 유성구청 AI 챗봇에서는 실사용자 인터랙션 흐름을 직접 설계했습니다. AI 서비스 특유의 비동기·스트리밍 UI 패턴에 익숙하며, 이를 안정적이고 재사용 가능한 컴포넌트로 구조화하는 데 강점이 있습니다.

### 수치로 증명하는 성능 최적화

측정 → 분석 → 개선 → 검증의 사이클을 반복하며, 모든 최적화 결과를 수치로 남깁니다. KISTEP 통합연구지원시스템에서는 Lighthouse 점수를 45에서 95로 끌어올렸고, 배포 시간을 5분에서 30초로, 개발 서버 로딩을 1분에서 1초로 단축했습니다. 온품의 Cesium 3D 지도 플랫폼에서는 JS Heap 메모리를 86% 줄이고, 3D Tiles 스타일 조건을 5,000개에서 20개로 99% 감소시켰습니다.

### 공통화·아키텍처 설계 능력

반복되는 설정과 구조를 조직 단위 보일러플레이트로 체계화해 팀 전체의 개발 기반을 정비해왔습니다. SimpleScaffold 프로젝트를 통해 React(Vite/Next), Micro Frontend, Nginx 미디어 서버, DB, 디자인 시스템까지 전 레이어를 커버하는 구조를 직접 설계했습니다. 특히 Micro Frontend 아키텍처는 Module Federation 기반의 Host·Remote·Shell 역할 분리, 공유 패키지 레이어(@repo/packages/fe-ui, fe-utils) 구성, 독립 배포 전략까지 꼼꼼히 구현했으며, 14개 섹션의 문서 체계로 정리해 처음 합류하는 팀원도 전체 구조를 파악할 수 있도록 했습니다. 이 노력으로 평균 배포 시간을 5분에서 30초로 단축하고, LLM 프로젝트 초기 도입 기간을 1달에서 2주로 줄였습니다.

### AI가 따를 수 있는 디자인 시스템 설계

디자인 감각이 없어도 공식을 따르면 일관된 결과물이 나오도록, 수치 기반 규칙으로 디자인 시스템을 문서화합니다. 색상은 6:3:1 비율과 HSL 팔레트로, 레이아웃은 8px 단위 그리드와 반응형 컬럼 체계(데스크톱 12 / 태블릿 8 / 모바일 4)로 정의합니다. 이 규칙들을 `.cursor/rules`에 탑재해 AI 코딩 도구가 새 컴포넌트를 만들 때도 시스템을 자동으로 준수하도록 설계합니다.

### 실전 보안 대응 경험

정보보안학과 전공과 K-Shield Jr. 수료를 바탕으로, 운영 서버에서 실제 침해 사고를 직접 발견하고 대응한 경험이 있습니다. 비정상적인 서버 부하를 인지한 후 SSH 로그 분석을 통해 무차별 대입 공격 경로를 파악했고, 의심 계정이 sudo·docker·lxd 권한을 악용한 사실을 확인했습니다. authorized_keys 백도어 제거, crontab 악성 항목 삭제, 악성 프로세스 종료를 거쳐 서버 포맷 및 검토된 데이터 선별 이전까지 전 과정을 직접 주도했습니다.

### 폭넓은 도메인 적응력

MLOps 데이터 관리, CCTV 관제, AI 챗봇, 3D 지도, 다차원 네트워크 시각화, 연구지원 시스템, 예산·회계 플랫폼까지 9개 이상의 전혀 다른 도메인 프로젝트를 소화했습니다. 도메인 지식이 없는 영역도 빠르게 흡수해 아키텍처 설계부터 배포까지 전 사이클을 담당해왔습니다.

## 성격 및 가치관

호기심이 많고 성실합니다. 새로운 기술과 방법론을 빠르게 흡수하는 편이며, 경험은 배신하지 않는다는 신념으로 다양한 도메인과 기술 스택에 열린 태도를 유지합니다. 최고의 기술보다는 상황에 맞는 기술을 선택하는 것을 중요하게 생각하고, 팀의 프로세스와 문화를 더 나은 방향으로 개선하려는 시도를 즐깁니다. 직관적인 UI와 세밀한 인터랙션에 관심이 많으며, 개발 환경과 코드 구조를 표준화하는 작업에서 보람을 느낍니다.

커뮤니케이션에서는 목표 정의 → 일정 산정 → 이슈 공유의 흐름을 중시합니다. 진행 상황을 적극적으로 공유하고 이슈를 빠르게 드러내는 방식으로 팀과 함께 일합니다.
