물리적 공간을 디지털로 복제한 디지털 트윈 기반 모니터링·분석·시뮬레이션 전문 기업.

### 주요 업무 및 성과

- **3D 기반 지도 플랫폼 설계 및 구현**
  - 3D 공간 데이터 시각화, 레이어 관리 구조 설계
  - 대용량 좌표 데이터 처리 및 렌더링 성능 최적화
- **지도 인터랙션 및 UX 개선**
  - 줌·이동·객체 선택 등 사용자 인터랙션 최적화
  - 데이터 가독성을 고려한 시각적 표현 및 UI 구성
- **확장 가능한 지도 아키텍처 구축**
  - 기능 모듈화로 관제·분석 시스템과 연계 가능하도록 설계
  - 신규 기능 추가 시 영향 범위를 최소화하는 구조 확립
- **리눅스 서버 침해 사고 발견 및 대응**
  - 비정상적 서버 부하·접근 이력을 직접 발견하여 침해 사고 인지
  - SSH 무차별 대입 공격 경로 확인, 의심 계정의 sudo/docker/lxd 권한 악용 사실 파악
  - 악성 계정 잠금, SSH authorized_keys 백도어 제거(immutable 속성 해제 포함), crontab 악성 항목 삭제, 악성 프로세스 종료 조치
  - /usr world-writable 권한 복구, 서버 포맷 및 검토된 데이터 선별 이전 권고
  - 관련 기록: https://ravenkim97.tistory.com/612

- **보일러플레이트 제작** → 평균 배포시간 5분 → 30초로 단축, 유지보수성 확보, 일관된 UI/UX 체계 확립

- **고성능 공통 컴포넌트 개발** → 30만 건 데이터까지 처리 가능한 테이블, 그래프, 파일 처리기 개발
- **다양한 도메인 경험** → MLOps 데이터 관리, CCTV 관제, AI 챗봇, 다차원 네트워크 시각화, 연구지원시스템, 인사관리 시스템
