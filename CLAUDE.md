## Overview

This is a GitHub profile repository (`ravenkim/ravenkim`). The `README.md` at the root appears publicly on the GitHub profile page. The rest of the repository is a personal document archive organized in Korean-named folders.

| 폴더 | 용도 |
|------|------|
| `이력서/` | 이력서 파일 및 외부 Google Docs 링크 |
| `경력/` | 회사별 경력 사항 |

## Conventions

- All folder names are in Korean.
- Each folder contains a `README.md` as its primary document.
- The root `README.md` is the public GitHub profile page — edits there are immediately visible on the profile.
