# 자기소개서

## 기본 소개

AI 도구를 적극 활용해 생산성을 높이는 React 프론트엔드 개발자 김해준입니다.

React를 주력으로 3년+ 실무 경험을 쌓았으며, AI 챗봇·LLM 시각화 등 AI 서비스의 프론트엔드를 직접 설계·구현해왔습니다. Claude Code·Cursor 등 AI 코딩 도구를 실무에 적극 도입해 반복 작업을 줄이고, 팀 전체의 개발 속도를 높이는 데 집중합니다. 구조적이고 확장 가능한 코드와 빠른 개발 사이클을 동시에 추구합니다.

## 핵심 강점

**1. AI 도구를 활용한 개발 생산성**
Claude Code·Cursor 등 AI 코딩 도구를 실무에 적극 도입해 반복 작업을 자동화합니다.
- AI 도구 기반 보일러플레이트 생성·유지 사이클 단축
- LLM 프로젝트 초기 도입 기간 1달 → 2주 단축
- 코드 리뷰·문서화·테스트 케이스 작성 등에 AI를 일상적으로 활용

**2. AI 서비스 프론트엔드 구현 경험**
LLM·AI 기반 서비스의 프론트엔드를 처음부터 설계하고 구현한 경험을 보유합니다.
- LLM 시각화 공통 템플릿(LimeLab): 스트리밍 응답 처리, 디자인 시스템 도입
- LLM VOC 플랫폼: Bllossom 모델 연동 기반 사용자 상호작용 처리
- AI 챗봇(유성구청): 업무 담당자 교체 시 인수인계 효율화

**3. 수치로 증명하는 성능 최적화**
측정 → 분석 → 개선 → 검증 사이클을 반복하며 결과를 수치로 남깁니다.
- KISTEP 통합연구지원시스템: Lighthouse 45 → 95, 배포 5분 → 30초, 개발 서버 로딩 1분 → 1초
- Cesium 3D 지도 플랫폼(온품): JS Heap −86%, 3D Tiles 스타일 조건 5,000개 → 20개 (−99%)

**4. 팀 생산성을 높이는 공통화·자동화**
반복되는 코드와 설정을 조직 단위 보일러플레이트(SimpleScaffold)로 체계화했습니다.
Frontend(Vite/Next), Micro Frontend, Nginx 서버, DB, 디자인 시스템까지 전 레이어를 커버합니다.

**5. 실전 보안 대응 경험**
정보보안학과 전공 + K-Shield Jr. 수료를 바탕으로, 실제 서버 침해 사고를 직접 발견하고 대응했습니다.
- 비정상적 서버 부하를 인지하고 SSH 로그 분석으로 무차별 대입 공격 경로 파악
- 의심 계정의 sudo/docker/lxd 권한 악용, authorized_keys 백도어, crontab 악성 항목 확인·제거
- 채굴 목적의 악성 프로세스 종료 및 서버 포맷·데이터 선별 이전 권고까지 전 과정 주도

**6. 폭넓은 도메인 적응력**
MLOps, CCTV 관제, AI 챗봇, 3D 지도, 네트워크 시각화, 연구지원, 예산·회계 등
9개+ 도메인 프로젝트를 소화하며 매번 새로운 환경에 빠르게 적응했습니다.

## AI 개발 방법론

단순 코드 생성이 아닌, **구조적 협업 방식**으로 AI를 활용합니다.

- **탐색 → 계획 → 실행 → 검증** 4단계 사이클 — AI와 함께하는 작업도 사람과 동일한 프로세스로 진행
- **Phase 분할** — 큰 작업을 단계별로 쪼개 컨텍스트 오염 없이 실행
- **규칙은 파일로 관리** — `.cursor/rules` 기반으로 팀 규칙을 AI에게 지속적으로 전달
- **성공 기준 먼저 정의** — "무엇을 만들지"보다 "언제 완성됐는지"를 먼저 설계
- **실제 동작 검증** — 브라우저 확인, 콘솔 에러, 기존 API 호환성까지 직접 확인

## 성격 및 가치관

- 호기심 많고 성실하며, 새로운 기술과 방법론을 빠르게 흡수합니다.
- 경험은 배신하지 않는다는 신념으로, 다양한 도메인과 기술 스택에 열려 있습니다.
- 재밌는 것과 함께 성장하는 것에 관심이 많습니다.

## 역량

**Overall**
- 직관적인 유저 인터페이스 및 애니메이션 구현을 좋아합니다.
- 회사 혹은 팀의 프로세스 및 문화를 개선하려는 시도를 적극적으로 합니다.
- 개발 환경, 코드 구조를 개선하거나 표준화하기를 좋아합니다.
- 최고의 기술보다는 상황에 맞는 기술을 선택하여 적용합니다.
- 업무에 필요하다면 익숙하지 않은 분야도 적극적으로 탐색합니다.

**Communication**
- 다양한 사회경험을 통해 커뮤니케이션에 능숙합니다.
- 일정 산정, 진행 상황 공유, 이슈 관리에 능숙하며 상황에 맞는 협업 툴을 활용합니다.
- 목표정의 → 시간 산정 → 이슈 공유 프로세스에 능숙합니다.
