## AI 개발 도구

| 도구 | 활용 방식 |
|------|----------|
| Cursor | 하네스 코딩 방식 적용 — `.cursor/rules` 기반 규칙 관리, Plan Mode 계획 수립, Phase 단위 실행 |
| Claude Code | 에이전트 루프 방식 — 탐색 → 계획 → 실행 → 검증 4단계 사이클 실무 적용 |
| GitHub Copilot | 코드 완성·리뷰·테스트 케이스 보조 |

> AI를 단순 코드 생성기가 아닌 **구조적 실행 파트너**로 활용합니다.  
> 큰 작업을 Phase 단위로 분할하고, 성공 기준을 먼저 정의한 뒤 AI와 협업합니다.
